# 配置文件

在TJAP3软件目录下往往都有一个文件名为config.ini，这是它的配置文件，其中包含了众多设置选项，以下介绍几种常用的设置。

由于TJAP3采用的文本编码均为Shift-jis，而它的配置文件也不例外，所以你需要准备一个顺手的文本编辑器，如果使用记事本软件打开则默认会采用GBK编码，会导致显示大量乱码。
这里推荐轻量级的樱花编辑器open in new window，或功能更丰富的Visual Studio Codeopen in new window。在打开config.ini时选择Shift-jis编码即可

# TJA路径

在config.ini中找到TJAPath，可填入曲包的路径，使多个模拟器可以共用同一个曲包。默认为整个模拟器目录。

如果你发现进入模拟器不读取曲包、或编辑了曲包内的TJA但并没有刷新时，可以尝试取消键盘的大写锁定。

# 字体

在config.ini中找到FontName，可修改为你想要的字体的名称。默认为微软雅黑
如果你使用的是OpenTaikoopen in new window，那么还可以设置BoxFontName，为分类文件夹的描述文字字体名称。

# 判定范围

在config.ini中找到如下语句，一般在中间至末尾处。这些是不同判定的区间，默认的判定范围如下，十分宽松。

遗憾的是，这些判定数值均为困难和魔王难度的的数字，并没有简单和普通数值可供设置。

[HitRange]

; Perfect～Poor とみなされる範囲[ms]
Perfect=36
Good=90
Poor=117
