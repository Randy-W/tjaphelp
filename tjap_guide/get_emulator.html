# 模拟器下载

可以从以下选择一个TJAPlayer3的分支来下载。

如果你不会从GitHub上下载，请点这里
  • 下图是一张GitHub的页面图片。除了位于中间的正文部分，右侧有一块名为Releases的区域。

  • 点击其中的版本号或下面的 + n releases ，可进入Releases页面

  • 在Releases页面可以看到发布的各个版本，在版本描述下面可以找到发布的文件，除了 Source code (zip) 和 Source code (tar.gz) 都是发布的版本，点击即可下载

# OpenTaikoopen in new window

  • 仿虹版的模拟器，同时也是少数修复了谱面分歧问题的模拟器。有中文说明文档。
  • 模拟器没有自带的皮肤，需要额外下载，建议前往其GitHub页面加Discord群下载或点击 这里open in new window 下载

皮肤作者：Discord用户あきら Ch.
关于皮肤的安装，请移步 相关资源 > 皮肤 详细查看

我怎么找不到最新的0.5.3Pre版啊？

版本号带Pre的版本是预发布版，不会直接显示在Releases区域。你需要点击下面的 + n releases
